# vault-setup-kit

Interactive bootstrap kit. Takes a fresh macOS device to a working Obsidian + Claude Code + AI-assistant operating system in one command.

Built for the RS42 founder's new devices, family trial users, and incoming RS42 interns.

## Usage

```bash
git clone git@github.com:RS42-AI/vault-setup-kit.git
cd vault-setup-kit
./install.sh
```

The wizard asks plain-language questions (areas, projects), then assembles the right building blocks into your vault.

## Power-user mode

```bash
./install.sh --non-interactive --areas=rs42:Work,personal:Personal --projects=hermes:rs42
```
